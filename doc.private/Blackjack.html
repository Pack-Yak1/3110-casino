type outcome =
| DealerBust
| Match of int 
val initial_cards : int

The number of cards every player is dealt at the start of a game.

val init_bet : bool

init_bet is true iff players are allowed to case bets before cards are dealt

val has_dealer : bool

has_dealer is true iff the game has a dealer controlled by the computer.

val win_check : outcome -> int -> bool

win_check outcome player_score is true if the player would win a game with outcome, otherwise it is false.
