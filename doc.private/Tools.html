val update_game : bool ->
       [> `Assoc of (string * [> `Int of int ]) list ] ->
       [> `Assoc of (string * [> `Int of int ]) list ]

update_game win g updates the number of wins and plays for game g. The number of wins is incremented iff win is true.

val update_if_match : string -> string -> string -> bool -> Yojson.Basic.t -> Yojson.Basic.t

update_if_match name money game win player is player with money set to to money and the number of wins and plays accurately incremented, in the game with name game, iff player has name name. Iff player has name name and win is true, the number of wins is incremented. Otherwise, return player.

val update_player_stats : string -> string -> string -> bool -> unit

update_player_stats name money game win sets the money to money and increments the number of plays of the game with name game for the player with name name. Iff win is true, the number of wins is incremented.

val update_total_game_stats : string -> unit
